# 💎 Diamond Prediction System

A Machine Learning web application that predicts:

- 💰 Diamond Price
- 💎 Diamond Cut Quality

## 🚀 Live Demo

https://diamond-vercel-app-3t5l-d1n1s0a4x-bhupinder07-sandhus-projects.vercel.app/

## 📊 Model Performance

### Price Prediction
- Algorithm: Linear Regression
- R² Score: 89.24%

### Cut Prediction
- Algorithm: Logistic Regression
- Accuracy: 99.88%
